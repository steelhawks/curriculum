# Learn CAD

Welcome to the Steel Hawks CAD curriculum. This is a structured walkthrough — start at the top and work your way down. Each topic builds on the one before it.

If you've never touched CAD before, don't worry. The first few lessons assume zero experience.

## How this is structured

- **Topics** are listed in the sidebar on the left. Each topic is a focused chunk of material.
- **Lessons** inside a topic are short and meant to be done in order.
- At the end of every lesson, there's a **Continue to next lesson** button.

## When you finish a topic

Talk to a lead designer. They'll either give you a small build to practice on, or move you to the next topic.
